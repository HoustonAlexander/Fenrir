# FENRIR REPORTER V.1
Batch processor for WRS P1 report generation at NCSU.

# FEATURES

3 Preset Reporting Intevals (Quarterly, Semi-Annually, or Annually)

Custom Date Range for any other reporting requirments

Loading Bar to track progress

Export Options for a more tailored experience

P3 Functionality Coming Soon!
File Consolidation Coming Soon!

# Requirements

NCSU Unity ID with Access to WRS

Python 3.8

tkinter

Pandas

Selenium
